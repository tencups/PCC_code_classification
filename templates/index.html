NHLBI PCC Assignment Tool (PCCAT)

Powered by NHLBI Chat - an OpenAI-based Large Language Model

Instructions

  1. Please save your branch’s PCC code and PO assignment spreadsheet to a “folder” of your preference.
  2. Select one spreadsheet by clicking [Choose Files].
  3. Click [Run] to generate PCC code.
  4. Click [View] to view the recommended PCC code in the same spreadsheet you saved in your “folder”.
